# 100 Days of DevOps Challenge

This repository documents my journey of completing the **100 Days of DevOps Challenge** by [KodeKloud](https://kodekloud.com/), led by **Mumshad Mannambeth**.

Each day covers practical DevOps topics such as **Linux, Git, Docker, Kubernetes, Jenkins, Terraform, and Ansible** with hands-on labs and examples.

---

### Topics Covered

- Linux & Git
- Docker & Kubernetes
- Jenkins CI/CD
- Terraform & AWS
- Ansible Automation
- Monitoring & Logging
